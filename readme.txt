Ciao, benvenuto nell'App Todolist.

Da terminale entra nella cartella dove hai clonato il progetto
Per installare le dipendenze lancia il comando => npm install
Successivamente per compilare il codice lancia il comando => npm run start
Accedi al link http://localhost:5000

Buon divertimento!

p.s. se la porta 5000 è già occupata, nel file main.ts puoi velocemente cambiarla.
